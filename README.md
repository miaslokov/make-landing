# Make.com No-code Landing Page

Landing page affiliate cho Make.com, tối ưu cho tệp người dùng no-code automation.

## Live URL
Sau khi deploy GitHub Pages:

- https://miaslokov.github.io/make-landing/

## Deploy
Trang được deploy tự động qua GitHub Actions mỗi lần push lên nhánh `main`.

## Files chính
- `index.html`: landing page chính
- `.github/workflows/deploy.yml`: workflow deploy GitHub Pages

## Lưu ý
- Đây là static site (HTML/CSS), không cần server.
- Có thể dùng custom domain sau nếu cần.
